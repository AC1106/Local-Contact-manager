##聯絡人管理程式
這個程式允許用戶添加、刪除、更新和查詢聯絡人信息。
所有的聯絡人信息將被儲存在本地文件中，並且會在程式啟動時加載，以便用戶能夠輕鬆管理他們的聯絡人。


##使用說明
啟動程式後，你會看到一個菜單，列出了可用的操作。
依照菜單的指示，選擇你想要進行的操作。
根據提示輸入相應的信息，完成操作。


##功能列表
添加聯絡人：將新的聯絡人信息添加到系統中。
刪除聯絡人：從系統中刪除指定的聯絡人。
更新聯絡人信息：更新指定聯絡人的信息，如電話號碼、電子郵件、城市和公司。
查詢聯絡人：通過姓名查詢指定聯絡人的信息。
儲存聯絡人數據：將所有聯絡人信息儲存在本地文件中，以便下次啟動時加載。
高級搜索聯絡人：通過姓名、電話、電子郵件、城市或公司進行高級搜索，並顯示匹配的結果。


##安全性注意事項
請注意，程式使用了對聯絡人信息進行加密和解密的功能，以確保用戶信息的安全性。請務必妥善保管程式中指定的密碼，並確保其足夠安全，以免造成數據泄露。
